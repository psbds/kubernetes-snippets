## Kubernetes Snippets CLI (cluster)

Set of DevOps Utilities for AKS

### Commands


* ```create-service-connection``` - Creates a new Service Connection on Azure DevOps to connect to a Kubernetes Cluster. 
 [See Docs.](https://github.com/psbds/kubernetes-snippets/tree/master/cli/devops/create-service-connection)
