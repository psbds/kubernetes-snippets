## Kubernetes Snippets CLI (cluster)

Cluster Management Tools

### Commands


* ```create``` - Create a new AKS Cluster. [See Docs.](https://github.com/psbds/kubernetes-snippets/tree/master/cli/cluster/create)
* ```create-aad-credentials``` - Creates the credentials on Azure Active Directory to integrate AKS with AAD Authentication. [See Docs.](https://github.com/psbds/kubernetes-snippets/tree/master/cli/cluster/create-aad-credentials)

### Arguments

| Arguments 	                | Type      | Default                   | Required  | Description 	|
|-----------	                |------     |---------                  |--------   |-------------	|
| -h, --help                    |           |                           | no        | Show help message.|
